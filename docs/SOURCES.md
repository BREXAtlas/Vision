# Sources and Factual Integrity

The typed registry in `src/content/sources.ts` contains reviewed official resources from FDIC, U.S. Treasury, Investor.gov/SEC, FINRA, SBA, NIST, CISA, HHS OHRP, IRS, FTC, USPTO, FAA, CFPB, and NIH. Each record includes organization, category, URL, review date, and lesson IDs.

Rules and figures change; verify current official guidance. Content is educational, not investment, financial, legal, tax, accounting, medical, or scientific advice. Estimates and vision values are labeled. Never infer clinical claims from an Atlas product concept.
