# Content Model

`Quarter`, `DailyChapter`, `QuizQuestion`, `Source`, `Project`, `Profile`, and `AppState` are strict TypeScript contracts. Seed input is date + display name + funding path. Hash selection is deterministic and draws from curated vocabulary, decisions, actions, future whispers, and visualizations. Content validation enforces minimum counts, valid answers, source IDs, coverage, determinism, and contiguous quarter boundaries.
